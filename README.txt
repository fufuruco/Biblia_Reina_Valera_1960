📖 SANTA BIBLIA - INSTRUCCIONES DE USO

============================================
🚀 CÓMO INICIAR LA APLICACIÓN
============================================

Opción 1: RECOMENDADA - Usar Python (más fácil)
─────────────────────────────────────────────
1. Haz doble clic en: iniciar_servidor.py
2. Se abrirá automáticamente en tu navegador
3. ¡Listo! Ya puedes usar la Biblia completa

(Si no tienes Python, descárgalo gratis desde python.org)


Opción 2: Usar Windows Batch
─────────────────────────────
1. Haz doble clic en: iniciar.bat
2. Se abrirá una ventana de comando
3. Abre tu navegador y ve a: http://localhost:8000/santa-biblia_html(1).html


Opción 3: Servidor manual (Node.js)
────────────────────────────────────
Si tienes Node.js instalado:
1. Abre PowerShell/CMD en esta carpeta
2. Ejecuta: npx http-server
3. Abre en el navegador: http://localhost:8080
